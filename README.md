**Doohickey** is a peer-to-peer platform that helps people rent out and borrow unused items, services, and spaces. It addresses the everyday problem of owning things you only need temporarily â€” by turning underutilized assets into income opportunities while reducing waste and clutter.

- Track inventory and manage personal belongings
